# Split Terminal 更新日志

## [1.2.0] - 2026-03-03

### 新功能
- 新建会话时增加项目选择对话框，可选择工作文件夹
- 支持从最近使用的项目列表中快速选择
- 支持浏览并选择任意文件夹作为工作目录
- 自动记录最近使用的项目（最多 10 个）
- 项目选择对话框支持快捷键操作（ESC 关闭）
- **文件夹栏跟随焦点会话自动切换**：当切换会话焦点时，左侧文件夹栏会自动显示该会话的工作目录

### 优化
- 新建会话按钮（+ 会话）现在会先弹出项目选择对话框
- 快捷键 Ctrl+Shift+N 改为打开项目选择对话框
- 新建标签页时也会弹出项目选择对话框
- 会话自动在选中的项目文件夹目录下启动
- **智能文件夹切换**：避免重复刷新同一目录，提升性能
- 每个会话独立存储工作目录（cwd），互不干扰

### 技术改进
- 增加项目选择对话框 UI 组件（HTML/CSS）
- 使用 localStorage 持久化最近使用的项目列表
- 添加路径验证功能，确保选择的文件夹存在
- 为每个会话添加 `cwd` 属性，记录工作目录
- 实现 `switchFolderForSession()` 函数，智能切换文件夹

## [1.1.0] - 2026-02-24

### 新功能
- 文件树支持 Ctrl+V 粘贴剪贴板文件到选中文件夹
- 文件树右键菜单增加「粘贴文件」选项

### 优化
- 修复中文输入法在终端中偶尔无法输入的问题（IME 组合状态检测）
- 减少文件树刷新频率和闪烁（fs.watch 仅响应 rename 事件 + 增量 DOM 更新）
- Claude 历史会话去重：同一会话链只显示一条，ID 使用最新续接会话（确保进入后看到完整内容）
- 大段文本粘贴分块写入（256字节/块，50ms间隔），避免内容丢失
- 分块写入时保护 ANSI 转义序列完整性，防止终端颜色错乱

### 修复
- 修复选中文件夹后 Ctrl+V 无响应的问题（fileTreeEl 焦点获取）

## [1.0.0] - 初始版本

### 功能
- 多分屏终端管理
- 文件树侧边栏（新建/重命名/删除文件夹和文件）
- Claude 历史会话列表（置顶/重命名/删除/快速进入）
- 定时任务功能（定时执行 Claude 指令）
- WebGL 渲染加速
- 会话状态持久化
